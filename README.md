Lipika

Lipika is a modern English ↔ Hindi Translator app built using Kotlin for Android.

Features

- English ↔ Hindi Translation
- Voice Input Support
- Modern & Smooth UI
- Navigation Drawer Menu
- Copy Translation
- Splash Screen
- Developer Instagram Integration
- Fast Translation API
